2) Responda as perguntas abaixo:


  1. Na tag <input> com atributo type="text", para que serve o atributo "placeholder"?


    2. Serve para definir o texto que aparecerá quando o input estiver vazio.


  2. Na tag <input> com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?


    3. Serve para deixar a opção pré-selecionada. Caso o usuário opte por não selecionar aquele input, basta "descheckar".


  3. Na tag <input> para que serve o atributo "value"?


    4. Serve para definir qual será a "id" da resposta para on servidor. Não afeta o usuário.


  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?


    5. Atributo "name", todos os radio do grupo devem possuir o mesmo "name".


  5. Para que serve a tag <label>? Para que serve o atributo "for" nessa tag?


    6. Serve para "etiquetar" como o próprio nome já diz. O "for" serve para endereçar a label a um input através de sua id.


  6. Na tag <option> para que serve o atributo "selected"?


    7. Serve para o mesmo fim que o "checked", para deixar uma opção pré-selecionada.


  7. Nas tags de formulário para que serve o atributo "required"?


    8. Serve para declarar que aquele input é NECESSÁRIO, que se o input não obtiver valor, o formulário não será enviado.


  8. Para que serve a tag <textarea>?


    9. Serve para inserir um input de texto que tem mais de uma linha.


  9. Na tag <form> para que serve o atributo "action"?


    10. O "action" serve para informar para onde vai o pacote de dados coletado no formulário.


  10. Na tag <form> para que serve o atributo "method"?


    11. Serve para especificar qual o método HTTP será aplicado no formulário


  11. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag <form>?


    12. Valor "POST"


  12. Qual valor deve ser informado no atributo "method" na tag <form> em um formulário que será usado para fazer upload de arquivos?


    13. Valor "GET"


  13. O que as tags <fieldset> e <legend> produzem?


    14. Fieldset agrupa elementos dentro de um formulário.
